Viral para Combate aos Mosquitos Aedes Aegypti

Este projeto contém o script para prover uma animação do mosquito aedes aegypti flutuante no site.

A proposta é que blogs, sites e outros serviços online, adotem o script abaixo como forma de viralizar o link para o site com orientações de como combater o mosquito aedes aegypti.
Para colocar em seu site, basta copiar e colar o código abaixo como conteúdo HTML
